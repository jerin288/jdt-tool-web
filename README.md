# JDT PDF to Excel Converter - Web Application

A web-based PDF to Excel converter with advanced extraction features, maintaining the same functionality as the desktop version but accessible through a browser.

## Features

- **PDF to Excel/CSV conversion** with multiple extraction modes
- **Advanced options:**
  - Page range selection (e.g., "1-3", "1,3,5", "all")
  - Extract tables, text, or both
  - Merge multiple tables
  - Header detection
  - Data cleaning
  - Password-protected PDF support
- **Modern web interface** with drag-and-drop support
- **Real-time progress tracking**
- **Responsive design** for mobile and desktop
- **Automatic file cleanup** for server efficiency
